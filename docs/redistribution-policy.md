# Redistribution Policy / 再配布方針

## English

This repository is primarily an English-language guide for SHARP PC-E500 series software.

Its purpose is to help users:

- discover software
- understand what each tool does
- prepare a working transfer environment
- reach the original distribution sources

This repository does **not** redistribute original archives unless explicit permission
for redistribution has been obtained from the relevant rights holder or author.

Until such permission is clearly confirmed, this repository will limit itself to:

- English documentation
- introduction pages
- setup guidance
- links to original sources

This policy is intended to respect original authors, original archive sites,
and the historical distribution context of the software.

---

## 日本語

このリポジトリは、SHARP PC-E500 系ソフトを英語圏利用者に紹介し、
導入を支援するための英語ガイド用リポジトリです。

目的は、利用者が

- ソフトの存在を知ること
- 各ツールの役割を理解すること
- 実際に使える転送環境を整えること
- 原本配布先にたどり着けること

にあります。

本リポジトリでは、権利者または作者から**明示的な再配布許諾が確認できるまで**、
オリジナルアーカイブの再配布は行いません。

許諾が明確になるまでは、以下に限定します。

- 英語ドキュメント
- 紹介ページ
- 導入ガイド
- 原本配布先リンク

これは、原作者・原本配布サイト・当時の配布文脈を尊重するための方針です。
