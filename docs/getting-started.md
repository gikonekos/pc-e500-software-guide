# Getting Started

This page is a practical starting point for users who want to begin using
software on the SHARP PC-E500 series.

It does not try to cover everything at once.
Its goal is to help new users understand the overall path.

## Basic idea

In many cases, getting started with PC-E500 series software involves more than
just downloading an archive.

You may need to:

- prepare a transfer environment
- handle older archive formats such as LZH
- use suitable communication tools
- pay attention to text encoding in practical transfer workflows

## Recommended order

A practical order is:

1. Read [Transfer Tools](transfer-tools.md)
2. Read [PLINK](../tools/plink/README.md)
3. Read [LZH Archives](lzh.md)
4. Check the software page you want to use
5. Obtain the original archive from the original distribution source
6. Prepare your transfer workflow carefully before sending files

## Important mindset

For historical software, the main difficulty is often not the software itself,
but the surrounding environment:

- connection hardware
- transfer tools
- archive extraction
- text encoding
- machine-specific procedures

If you prepare those parts first, the software becomes much easier to use.

## Original archives

This repository is a guide repository.

has been obtained.

Please check each software page for original distribution information.
