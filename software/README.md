# Software

This directory contains software-specific pages for the SHARP PC-E500 series.

The focus here is on individual software titles, development tools, utilities,
and other notable programs in the PC-E500 software ecosystem.

Each subdirectory is intended to provide an English-language entry point for one
software package or software family.

Pages in this directory may include:

- a general introduction
- what the software does
- why it matters
- compatibility notes
- links to original distribution sources
- translated documentation where appropriate

This directory is not intended as a redistribution archive by default.

Please check each individual software page for:

- original distribution information
- documentation links
- usage notes
- redistribution status
