# PC-E500 Software Guide

English guide and introduction repository for SHARP PC-E500 series software.

This repository is intended to help English-speaking users discover, understand,
and start using software for the SHARP PC-E500 series.

Its main purpose is:

- software introductions in English
- setup and transfer guides
- links to original distribution sources
- practical notes for getting started on real hardware

This is **not** a redistribution repository.

Original archives are **not rehosted here unless explicit permission has been obtained**.
Please use the links to the original distribution sources whenever possible.

---

## Scope

This repository focuses on:

- SHARP PC-E500 series related software
- English introduction pages
- installation and transfer guidance
- notes for old archive formats such as LZH
- practical getting-started documentation

This repository does **not** aim to replace original archive sites or original authors.

---

## First Steps

If you are new to the PC-E500 ecosystem, start here:

1. Read the introduction to the transfer environment
2. Learn what PLINK is and why it matters
3. Prepare a Windows-side transfer tool
4. Learn how to handle LZH archives
5. Follow the getting-started guide
6. Explore individual software pages

Some tools may require serial connection hardware, transfer software,
or machine-specific setup.

This repository tries to make those steps easier for new users.
